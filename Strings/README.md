# 🔤 Strings – DSA in C

This folder contains string manipulation programs in C.

## 📌 Programs Included

* Palindrome check
* Reverse string
* Count vowels, consonants, spaces
* Compare strings (without strcmp)
* Anagram check
* Remove duplicate characters

## 📖 Concepts Covered

* Character arrays
* String traversal
* Frequency counting (hashing)
* Basic string algorithms

## 🎯 Learning Outcome

* Improve string manipulation skills
* Build logic for interview questions
* Understand character-level operations
