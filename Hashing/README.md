# 🔐 Hashing – DSA in C

This folder contains basic hashing implementations in C.

## 📌 Programs Included

* Hash table insertion (linear probing)
* Hash table search
* Hash table deletion
* Frequency counting using hashing

## 📖 Concepts Covered

* Hash function
* Collision handling (Linear Probing)
* Open addressing
* Frequency counting

## 🎯 Learning Outcome

* Understand how hash tables work
* Learn collision resolution techniques
* Apply hashing to solve problems efficiently
