# 🌳 Trees – DSA in C

This folder contains implementations of **Tree Data Structures** in C as part of my **DSA Phase 2 (Intermediate Level)**.

Trees are a hierarchical data structure and are widely used in searching, sorting, and hierarchical data representation.

---

## 📌 Programs Included

### 🔹 Binary Tree

* Create Binary Tree (Recursive)
* Inorder Traversal (Left → Root → Right)
* Preorder Traversal (Root → Left → Right)
* Postorder Traversal (Left → Right → Root)
* Level Order Traversal (BFS)
* Count total nodes
* Find height of tree
* Search element in Binary Tree

---

### 🔹 Binary Search Tree (BST)

* Create & Insert in BST
* Delete node from BST
* Search element in BST

---

## 📖 Concepts Covered

* Tree data structure
* Recursion
* Depth First Search (DFS)
* Breadth First Search (BFS)
* Binary Search Tree operations
