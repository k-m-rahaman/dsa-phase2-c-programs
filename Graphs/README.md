# 🌐 Graphs – DSA in C

This folder contains basic graph implementations in C.

## 📌 Programs Included

* Graph using adjacency matrix
* Graph using adjacency list
* Breadth First Search (BFS)
* Depth First Search (DFS)

## 📖 Concepts Covered

* Graph representation
* Traversal algorithms
* Queue (for BFS)
* Recursion (for DFS)

## 🎯 Learning Outcome

* Understand graph data structures
* Learn traversal techniques
* Build foundation for advanced graph algorithms
